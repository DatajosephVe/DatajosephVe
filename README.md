## 📂 Proyectos

### 📦 Brz Ecommerce Analytics
> SQL Server · Power BI · DAX · T-SQL

Análisis de punta a punta sobre un marketplace brasileño real con +100,000 órdenes.  
Desde el modelado relacional en SQL Server hasta el dashboard interactivo en Power BI.

**Lo que incluye:**
- 🗄️ Base de datos relacional con PK/FK, normalización e índices compuestos
- 👁️ 6 vistas analíticas reutilizables como capa semántica
- 🔍 18 consultas de negocio sobre clientes, ventas, logística y satisfacción
- 📊 Dashboard interactivo en Power BI con modo claro y oscuro

**Métricas del proyecto:**
`+100K órdenes` · `$8.7M en ventas` · `90.8% SLA` · `4.09/5.00 satisfacción`

### 🛒 SuperStore Sales Analytics
> Excel · Power Query · Power Pivot · DAX

Análisis de 4 años de ventas de una tienda retail americana (2014–2017) construido completamente dentro del ecosistema Excel.

**Lo que incluye:**
- 🔄 ETL completo con Power Query — 5 consultas de transformación
- 🏗️ Modelo estrella en Power Pivot con tabla calendario
- 📐 30 medidas DAX incluyendo YoY, acumulados y rankings
- ⚠️ Detección de anomalías con Z-Score implementado en DAX

**Métricas del proyecto:**
`9,994 transacciones` · `$2.3M ventas` · `12% margen` · `3 anomalías detectadas`

[![Ver Repositorio](https://img.shields.io/badge/Ver_Repositorio-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/DatajosephVe/SuperStore-Sales-Analytics)
